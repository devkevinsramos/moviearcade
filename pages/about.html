O que é o MovieArcade?

MovieArcade é um projeto front-end criado para transformar um catálogo de filmes em uma experiência mais interativa. A ideia do site é unir curadoria, listas, dados de filmes e pequenos arcades em uma interface simples, responsiva e divertida.

O projeto foi desenvolvido com HTML, CSS e JavaScript, pensando em uma navegação leve para web e em uma estrutura compatível com o GitHub Pages. Em vez de funcionar apenas como uma lista comum de filmes, o MovieArcade propõe formas diferentes de explorar o catálogo, como roleta, duelos entre filmes e quizzes.

A proposta combina curadoria, listas, dados de filmes e minigames em uma interface responsiva feita para web.

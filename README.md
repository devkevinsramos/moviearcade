# MovieArcade

A personal front-end project that combines cinema, visual design, and interactive movie-based experiences.

Instead of being only a movie catalog, MovieArcade is built as an interactive website with curated movies, lists, quizzes, roulette, versus mode, and a strong visual identity inspired by cinema.

Live preview: https://devkevinsramos.github.io/moviearcade/index.html

## About

MovieArcade was created to explore front-end development through a project with a clear visual style and interactive features.

The idea is to build a movie-focused website where users can explore selected films in a more fun and dynamic way. The project mixes curated content, movie data, lists, and small arcade-style experiences.

It is also a practical study of layout, responsiveness, JavaScript logic, API usage, and UI/UX decisions for the web.

## Features

The project includes:

- landing page
- home page with featured sections
- movie catalog with selected films
- movie lists by genre and theme
- quiz section
- movie roulette
- versus mode
- about page
- responsive layout
- interactive JavaScript features
